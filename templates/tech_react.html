React

Overview

React is a JavaScript library developed by Facebook for building user interfaces. It uses a component-based architecture, allowing developers to create reusable UI components and manage the application's state effectively.

React makes it painless to create interactive UIs. Design simple views for each state in your application, and React will efficiently update and render just the right components when your data changes.

Key Features of React

  • Component-based Architecture: Build encapsulated components that manage their own state, then compose them to make complex UIs.
  • Virtual DOM: React creates an in-memory data structure cache, computes the resulting differences, and then updates the browserâ€™s displayed DOM efficiently.
  • One-way Data Binding: Ensures that data flows in a single direction, making the application more predictable and easier to debug.
  • Strong Community and Ecosystem: A large ecosystem of libraries and tools that work well with React, such as Redux for state management, React Router for navigation, and many more.

Personal Kanban Board

Developed a Personal Kanban board application using React, TypeScript, and Firebase. This project helps users manage their tasks efficiently with features like drag-and-drop, data persistence, and cloud storage.

  • React: Utilized React for building a dynamic and responsive user interface with component-based architecture.
  • TypeScript: Leveraged TypeScript for static typing, which improves code quality and maintainability.
  • Drag-and-Drop: Implemented drag-and-drop functionality to allow users to move tasks between different stages (To Do, Doing, Done).
  • Firebase: Integrated Firebase for real-time database functionalities, user authentication, and cloud storage.
  • Data Persistence: Ensured task data is saved and synchronized using Firebase Firestore.
  • Archive and Trash Boards: Added separate boards for archiving completed tasks and managing deleted tasks.
  • Search and Filter: Enabled users to search and filter tasks for better task management.
  • Dark Mode: Implemented a dark mode feature to enhance user experience and reduce eye strain in low-light environments.
